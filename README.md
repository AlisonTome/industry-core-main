# NexForge — Marketplace de Projetos Técnicos

Plataforma B2B para conexão entre indústrias que precisam de fabricação e fornecedores técnicos qualificados.

## 🚀 Acesso

Site publicado via GitHub Pages:  

**https://AlisonTome.github.io/industry-core-main**


## 🗂 Estrutura do projeto

```
nexforge/
├── index.html   → Estrutura HTML (páginas, formulários, dashboards)
├── style.css    → Estilos, variáveis de cor e layout
├── app.js       → Toda a lógica da aplicação (auth, estado, renders)
└── README.md    → Este arquivo
```

## ⚙️ Tecnologias

- HTML5 + CSS3 (variáveis CSS, grid, flexbox)
- JavaScript vanilla (sem frameworks)
- Google Fonts (Syne, DM Sans, DM Mono)
- Estado em memória (sem backend)
